oc index

  • bxinorm: weighted norm (for iscarc)
  • cbcf: BC for OC with iscnat; require ya=u0 and psi*(yb-u1)=0
  • cbcfe: (extended) BC for OC with iscarc;
  • cbcjac: Boundary jacobian for OC with iscnat;
  • cbcjace: Boundary jacobian for OC with iscarc (extended)
  • disjca: discount jca (use for CSS)
  • fjac: Jac of rhs for OC BVP (in t)
  • fjace: Jac of rhs for OC BVP (in t) solver (iscarc)
  • getPsi: get matrix Psi for projection on E_u of target CSS
  • iscarc: Initial State Continuation ARClength (parametrization)
  • iscnat: Initial State Continuation NATural (parametrization)
  • jca: integrate J_c in space and normalize by p.vol
  • jcai: jca along canonical path
  • loadcp: convenience function to load CP
  • mrhs: rhs for OC BVP (in t) solver
  • mrhse: (extended) rhs for OC BVP (in t) solver
  • ocbra: standard output to bifurcation diagram for OC
  • ocstanopt: set standard options for OC problems
  • psol3D: OC version of x-t plots
  • setfnflip: convenience function to fill fn with dir and ptnames of CSS
  • simps: simpson rule integration by Damien Garcia
  • spcalcoc: OC version of spcalc, calculate (gen) EVals/EVecs of Gu'.
  • stancssvalf: print CSS value and characteristics
  • standiagn: plot u_1, lam_1 for OC; use as template to plot adapted diagnostics
